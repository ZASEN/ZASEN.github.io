vim基础语法与操作

一、基本语法

  1. vi -r：恢复崩溃时正在编辑的文件

  2. vi -R：以只读方式打开文件

  3. vi +n：打开文件时光标在第n行，若不指定则默认最后一行

二、基本模式

  1. 命令模式

  2. 编辑模式

  3. 末行模式

  4. 模式间切换
    （1）命令模式 → 编辑模式：有6种方式

    （2）编辑模式 → 命令模式：esc键
    （3）命令模式 → 末行模式：冒号
    （4）末行模式 → 命令模式：esc键

三、基本操作

在命令模式下：

  1. 连按2次d：剪切整行内容
    在dd前加数字，表示删几行，如2dd，表示删2行内容

  2. p：粘贴内容

  3. /+搜索内容：搜索关键字，按n键向后遍历

  4. ?+搜索内容：搜索关键字，按n键向前遍历

在末行模式下：

  1. w：保存

  2. q：退出

  3. !：强制执行

  4. 替换关键字

    如%s/lcl/zxy/g,表示将原文中lcl的内容替换为zxy，/g表示不提示直接执行

  5. 若要指定a行到b行间替换

    如1,5s/lcl/zxy/g,表示将原文第1行到第5行中lcl的内容替换为zxy
